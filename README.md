### Why do this？

一手的、准确的技术资料基本都是英文的，想要很好的理解这些内容要求程序员具备良好的英文水平，然而像我这样学渣出身的程序员这本身就是一件有挑战的事情。

尝试了百词斩、AnkiDroid 和扇贝等记单词的软件，比较喜欢百词斩的风格，但是只有四级、六级、托福和雅思这样的分类，并没有程序员专用的单词列表。

所以为了提升学习英语的效率，我想定制一份程序员专用单词列表。
